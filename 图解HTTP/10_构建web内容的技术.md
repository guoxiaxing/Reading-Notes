## 与 Web 服务器及程序协作的 CGI

CGI（Common Gateway Interface） 通用网关接口。是指 Web 服务器在接收到客户端发送过来的请求后转发给程序的一组机制。在 CGI 的作用下，程序会对请求内容做出相应的动作，比如创建 HTML 等动态内容。
